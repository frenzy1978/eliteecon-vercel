# EliteEcon Calibration Workflow

This folder holds benchmark scripts and teacher reference marks used to calibrate marking quality.

## Files
- `benchmark-template.json` — starter structure for benchmark entries.
- `benchmark-sample.json` — sample filled entry.
- `runbook.md` — step-by-step calibration process.

## Process summary
1. Add anonymized responses to benchmark JSON.
2. Add teacher reference judgement for each.
3. Run local comparison script.
4. Review deltas and tune prompts/rules.
5. Repeat weekly.
