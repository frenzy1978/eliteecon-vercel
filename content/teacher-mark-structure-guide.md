# EliteEcon Source Guide — Teacher Structure Notes

Provided by Mick for A-Level Economics students.
Use this as a guidance source for prompt tuning and student-facing feedback.

## Why structure matters
- Structure = clear thinking = higher marks
- Helps examiners follow reasoning
- Shows AO1/AO2/AO3/AO4 clearly

## Assessment Objectives
- AO1: Knowledge (definitions, theory)
- AO2: Application (real-world context)
- AO3: Analysis (chains of reasoning, diagrams)
- AO4: Evaluation (weighing arguments, judgement)

## 9-mark questions
- AO1–AO3 only (no evaluation)
- Usually Explain/Analyse
- Focus: clarity, logic, one strong diagram

Suggested structure:
1. Define key term(s)
2. Explain mechanism/process
3. Apply to real-world example
4. Analyse with diagram + chain of reasoning

Tip: 2 short paragraphs + 1 diagram

## 10-mark questions
- Slightly deeper than 9-mark
- Can include evaluation depending on command word
- Two developed paragraphs expected

If no evaluation:
- Define -> Explain -> Apply -> Analyse

If evaluation required:
- 1 analysis + 1 evaluation paragraph
- PEEL (+ However)

## 15-mark questions
- Mainly AO1–AO3
- Explain/discuss concept in context
- Depth/coherence > length

Suggested structure:
- Intro: define + focus
- 3 PEEL paragraphs (theory, application, second cause)
- One well-labelled diagram
- Brief conclusion summarising mechanisms

## 25-mark questions
- Essay style AO1–AO4
- Evaluate/Assess/To what extent/Discuss
- Requires balance and judgement

Suggested structure:
- Intro: define key term + outline both sides
- Main body: 3 analysis + 2 evaluation paragraphs
- Conclusion: overall judgement + conditions
- Include 1–2 well-drawn diagrams

Evaluation starters:
- However, this depends on...
- In the short run / long run...
- If elasticity is high/low...
- Government failure may occur if...

## Diagrams
- Label axes/curves clearly
- Identify equilibrium and welfare loss where relevant
- Keep neat; diagrams support AO3

## Time management guidance
- 9 mark: 10–12 mins
- 10 mark: 12–15 mins
- 15 mark: 18–20 mins
- 25 mark: 30–35 mins

## Quick summary
- 9 mark: AO1–3 | Define -> Explain -> Apply -> Analyse | No eval
- 10 mark: AO1–3 (+AO4 sometimes) | PEEL x2–3 | Mini eval sometimes
- 15 mark: AO1–3 | 3 PEEL + diagram | No eval
- 25 mark: AO1–4 | 3 analysis + 2 eval + judgement | Yes eval

## Final advice
- Define key terms
- Use at least one diagram for analysis
- Apply to real-world contexts
- Link back to question every paragraph
- Use evaluation language for balance
